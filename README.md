## PPG2_2526-ZeldaLike
Mini-jeu 2D JavaFX développé à l’EPFC dans le cadre du cours de Programmation Orientée Objet (Java 2). 
Le projet évolue par itérations : 
- Création du monde
- Conception du personnage
- Déplacements
- Detection des collisions 
- Correction des collisions
- Encapsulation progressive des classes selon les principes POO.
